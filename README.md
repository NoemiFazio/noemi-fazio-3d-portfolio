# 3D Portfolio

A 3D portfolio project built with **React**, **Three.js**, and **GSAP**.

## Features

- Complex animations using GSAP
- Interactive 3D scenes with Three.js and @react-three/fiber
- Optimized build with Vite
- Linting with ESLint + Prettier
- Husky + lint-staged for clean commits

## Technologies

- React 19
- Three.js 0.181
- GSAP 3.13
- TypeScript
- Vite

## Installation

```bash
pnpm install
pnpm dev
```
